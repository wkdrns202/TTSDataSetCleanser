# TTS Dataset Cleanser

긴 음성 데이터를 TTS(Text-to-Speech) 학습용 데이터셋으로 변환하는 도구입니다.

## 사용법

1. 오디오 파일 준비: 
   - 원본 오디오 파일(mp3, wav 등)을 `raw_audio/` 폴더에 넣고 작업을 진행하면 됩니다. 장시간 분량의 원본 오디오도 대응가능 하지만 긴 작업시간이 필요하므로 추천하지 않습니다.

2. 설치:
   ```bash
   pip install -r requirements.txt
   ```
   *(참고: 시스템에 FFmpeg가 설치되어 있어야 합니다.)*

3. 실행:
   - 추후 제공될 스크립트를 통해 오디오 분할 및 라벨링을 수행합니다.

4. 결과물 평가:
   - 오류율을 report 해줍니다.
